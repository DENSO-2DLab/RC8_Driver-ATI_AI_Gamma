# RC8 Driver - ATI AI Gamma
RC8 Driver for the ATI AI Gama force torque sensor

### Contents

RC8 Driver for reading force/torque data from ATI AI Gamma sensor via TCP/IP. 

### System Requirements

- RC8 Controller
- TCP/IP Communication between Robot and Sensor
- ATI F/T Sensor
